# Simple Calendar App

Repository skeleton for the Simple Calendar App (Vanilla JS + date-fns). This repo follows the project's SDD/TDD flow: `speckit.constitution` -> `speckit.spec.md` -> `plan.md` -> `tasks/*`.

See `tasks/calendar-tasks.md` for the full task breakdown and next steps.

Note: No dependencies have been installed by the bot. Run `npm install` locally when ready.
